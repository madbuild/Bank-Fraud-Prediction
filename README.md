# Bank-Fraud-Prediction
Fraud detection with machine learning in Python



This project uses machine learning to detect fraudulent banking transactions.  
It shows how data science can help identify suspicious activity and reduce financial risk.

## About the project
- Works with real transaction data  
- Handles class imbalance in fraud datasets  
- Builds classification models (Logistic Regression, Random Forest, XGBoost)  
- Evaluates performance with metrics like Precision, Recall, F1-score, and ROC-AUC  
- Provides a reproducible Jupyter Notebook workflow  

## Tech stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn. 

---

The goal is to give a clear, end-to-end example of how fraud detection systems can be built using machine learning.
